# Robot Modeling and Simulation

This repository contains projects developed during the Modeling course of the Robotics Engineering degree.

The projects focus on robot modeling, simulation, and physics-based environments using Python, URDF, Blender, and PyBullet.

## Contents

Practice 1:
Simulation of a mobile robot using PyBullet and URDF models.

Practice 2:
Robot modeling using Blender, URDF, and physics simulation.

Practice 3:
Simulation workspace for robot modeling experiments.

## Technologies

- Python
- PyBullet
- URDF
- Blender
- Robotics simulation

## Note

All code was developed for academic purposes.
