# Robot Modeling and Simulation

This repository contains projects developed during the Modeling course of the Robotics Engineering degree.

The projects focus on robot modeling, simulation, and physics-based environments using Python, URDF, Blender, and PyBullet.

## Contents

### Practice 1 — PyBullet Simulation

Simulation of a mobile robot using PyBullet and URDF models, including environment interaction and result analysis.

### Practice 2 — Robot Modeling

Robot modeling using Blender and URDF, including mesh generation and physics-based simulation.

### Practice 3 — ROS & Gazebo Simulation

A complete robotic simulation developed using ROS and Gazebo.

🔗 Repository:
https://github.com/ana-martinezal2021/Practica_3-Simulacion_en_ROS_y_Gazebo
